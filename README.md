# income-calculator

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```



## 思路

```tex
使用props和emit进行父子组件间通信，然后计算出值

通过遮罩弹出用户信息和设置，进行修改

修改设置或用户信息后，重新调用计算方法
```
